    - When threat level changes (escalates), add a brief CSS animation — a pulse/flash effect using Tailwind `animate-pulse` for 2 seconds then stop. Use a useEffect that detects level changes and applies a temporary class.
    - Font: bold, tracking-wide, text-sm or text-base

  <action>
    Create the map panel — the centerpiece of the dashboard.

    **useVesselPositions hook** (`frontend/src/hooks/useVesselPositions.ts`):
    - On mount, fetch all vessel positions from `vessel_positions` table, ordered by timestamp desc
    - Group by MMSI — keep only the latest position per vessel
    - Subscribe to realtime INSERT events on `vessel_positions`
    - When new position arrives, update the vessel's position in state (latest position per MMSI)
    - Return: `{ vessels: Map<number, VesselPosition>, loading: boolean }`

    **MapPanel component** (`frontend/src/components/MapPanel.tsx`):
    - Use `react-map-gl` with Mapbox `mapbox://styles/mapbox/light-v11` style
    - NOTE: Mapbox requires an access token. Use an environment variable `VITE_MAPBOX_TOKEN`. In the component, read `import.meta.env.VITE_MAPBOX_TOKEN`. If token is not set, render a fallback: a light gray div with a simple SVG outline of the Taiwan Strait (lat 23-26, lon 118-122) and text "Map: Set VITE_MAPBOX_TOKEN for Mapbox". This ensures the app works without a Mapbox account.
    - Fixed view centered on Taiwan Strait: `latitude: 24.5, longitude: 120.0, zoom: 6.5`
    - Disable all interaction: `scrollZoom={false}`, `dragPan={false}`, `dragRotate={false}`, `doubleClickZoom={false}`, `touchZoomRotate={false}`
    - Render vessel markers using `<Marker>` components for each unique vessel:
      - Icon: A small triangle/arrow shape (CSS or inline SVG) rotated by `course` degrees to show heading
      - Color: `text-blue-600` for normal vessels
      - Size: ~16px
      - On new position update, add a brief CSS `animate-ping` effect (single ping, not continuous) to indicate the update — use a state flag that clears after 1 second
    - Tooltip on hover: Show ship_name, speed, and course

    **useMovementHeatmap hook** (`frontend/src/hooks/useMovementHeatmap.ts`):
    - On mount, fetch all movement_events from `movement_events` table, selecting `location_lat`, `location_lon`, and `event_type`
    - Filter out rows where location_lat or location_lon is null
    - Subscribe to realtime INSERT events on `movement_events`
    - Return: `{ heatmapPoints: Array<{ lat: number, lon: number, type: string }>, loading: boolean }`
    - NOTE: This hook is separate from 04-03's `useMovementEvents.ts` (which does temporal matching for post cards). This hook provides only location data for the heatmap layer.

    **useNarrativeRegions hook** (`frontend/src/hooks/useNarrativeRegions.ts`):
    - On mount, fetch all narrative_events from `narrative_events` table, selecting `event_type`, `summary`, `detected_at`
    - Subscribe to realtime INSERT events on `narrative_events`
    - Extract geographic focus from event text: scan each event's `summary` and `event_type` for known geographic keywords. Use a predefined mapping of geographic terms to bounding regions:
      - "Taiwan Strait" / "strait" / "taiwan" → polygon: `[[117.5, 22.5], [121.5, 22.5], [121.5, 26.0], [117.5, 26.0]]` (lon/lat bounding box for the strait)
      - "South China Sea" / "south china" → polygon: `[[110.0, 5.0], [121.0, 5.0], [121.0, 22.0], [110.0, 22.0]]`
      - "East China Sea" / "east china" → polygon: `[[120.0, 25.0], [130.0, 25.0], [130.0, 33.0], [120.0, 33.0]]`
    - Return: `{ activeRegions: Array<{ name: string, polygon: number[][], eventCount: number }>, loading: boolean }`
    - Deduplicate: if multiple events reference the same region, return it once with accumulated eventCount
    - For the demo, this simple keyword-to-polygon approach is sufficient. No geocoding API needed.

    **Heatmap layer in MapPanel**:
    - Consume `useMovementHeatmap` hook
    - When using react-map-gl (Mapbox token is set):
      - Add a Mapbox `heatmap` layer using `<Source>` and `<Layer>` components from react-map-gl
      - Create a GeoJSON FeatureCollection from heatmapPoints (each point as a Feature with Point geometry)
      - Heatmap layer config: `heatmap-radius`: 20, `heatmap-opacity`: 0.6, `heatmap-intensity`: 1, `heatmap-color` interpolation from transparent → blue → cyan → yellow → red
      - Layer renders below vessel markers (use `beforeId` to order layers)
    - When using fallback (no Mapbox token):
      - Render heatmap points as small semi-transparent colored circles on the SVG fallback map, sized proportionally to clustering density (simple approach: just plot each point as a small dot)

    **Narrative focus regions in MapPanel**:
    - Consume `useNarrativeRegions` hook
    - When using react-map-gl (Mapbox token is set):
      - For each activeRegion, render a `<Source>` with GeoJSON Polygon and a `<Layer>` of type `fill` with semi-transparent fill: `fill-color: "rgba(147, 51, 234, 0.15)"` (purple tint), `fill-outline-color: "rgba(147, 51, 234, 0.4)"`
      - Render below vessel markers but above the heatmap layer
    - When using fallback (no Mapbox token):
      - Render the bounding regions as semi-transparent SVG `<rect>` or `<polygon>` elements on the fallback map
    - If no narrative events exist (empty array), render no region overlays — this is the expected initial state

    **Wire into DashboardLayout**: Replace the map placeholder with `<MapPanel />`.

    WHY: User specified clean minimal base map (Mapbox Light), directional ship/arrow icons, semi-transparent shaded regions for narrative focus areas, and fixed view locked to Taiwan Strait. The heatmap overlay shows movement cluster density from movement_events locations. The narrative regions show where state media narratives are geographically focused. The fallback ensures the demo works without a Mapbox API key.
  </action>

If movement_events with location data exist, a heatmap layer renders showing density clusters. If narrative_events exist with geographic keywords (e.g., "Taiwan Strait" in summary), semi-transparent purple shaded regions render on the map. `npx tsc --noEmit` passes. If Mapbox token is set, map shows light style centered on Taiwan Strait with no ability to pan/zoom, heatmap layer visible beneath vessel markers, and narrative regions as purple overlays.
